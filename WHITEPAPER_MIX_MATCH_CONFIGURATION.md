## Abstract

**Mix and match** — pick the arm for the task. Any combination. Each node station accepts 6, 8, 12+ arms. Combine pick-and-place, welding, assembly, packaging in one station. **Post-singularity configurable** — no full reprogramming to change arm mix. Natural language task description drives which arms activate. This whitepaper specifies the mix-and-match model and reconfiguration protocol.

---

## Multiarm Specialized Nodes

- **Per station:** 6, 8, 12+ arms — configurable
- **Per arm:** Seed:edge pair; one task type (pick-place, weld, assemble, pack)
- **Per station mix:** Any combination — e.g., 4 pick-place + 2 weld + 2 assembly

---

## Reconfiguration Without Full Reprogramming

Describe new task → edge point set → system auto-generates to seeds → selects appropriate arms. Mix-and-match means the station topology stays; arm assignment per task changes via SING! layer. Kerrigan orchestrates; Nevatio hardware executes.

---

## Differentiation

Traditional automation: fixed arm assignments; reprogram for new mix. SING! Omnispin: mix-and-match; describe it → execute. Post-singularity configurable.
