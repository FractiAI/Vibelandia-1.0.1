- **Design:** VIBE symbol as stamp design
- **Size:** Standard postage stamp dimensions
- **Colors:** Brand colors (purple, blue, cyan gradient)

- **Placement:** Inside garment (jackets, hoodies, hats)
- **Design:** VIBE symbol + "Designed by Hero Host Gianni" + brand mark
